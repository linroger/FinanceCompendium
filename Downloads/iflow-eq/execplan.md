# Financial Mathematics Equations Dataset - ExecPlan

## Project Overview
**Objective**: Create a comprehensive, MECE dataset of 300-500 equations used in financial mathematics and financial instruments pricing, including risk metrics and related attributes.

**Scope**: Cover all major categories of financial mathematics including basic finance math, bonds, stocks, options, futures, swaps, risk metrics, portfolio theory, derivatives, interest rate theory, corporate finance, quantitative finance, financial modeling, and statistical methods.

**Output Format**:
- CSV dataset with columns: title, equation (LaTeX without delimiters), main category, notes
- Markdown primer document explaining the dataset and usage
